BiLang RIS Abschlussarbeit 2021

Final Project: Automated Translation Middleware for JTL-Ecosystem

For my final examination as a Software Developer, I designed and implemented a scalable solution to automate multilingual content management within the JTL e-commerce ecosystem. The project involved developing a C#/.NET desktop application and a RESTful API built with PHP and the Laravel framework.

Key Features & Technologies:

  • Automated Translation Pipeline: Integrated the DeepL API to provide high-quality, automated translations for product data, categories, and SEO content, significantly reducing manual effort and preventing data inconsistency.
  • Middleware Architecture: Developed a custom Laravel-based API to handle communication between the shop systems and translation services, including usage metrics and billing logic.
  • Data Integrity: Implemented secure data handling for MSSQL databases using the JTL-Ameise interface to ensure compatibility and stability within the Windows-based ERP environment.
  • Full-Stack Development: Managed the project from conceptualization to implementation, utilizing MVC patterns, MySQL, and JSON-based communication to create a maintainable and extensible software architecture.
